# TI-84 POC

A simple multiplayer game on the TI-84 graphing calculator

# Setup

## Requirments

- 2 TI-84 graphing calculators
- A cable for I/O port
- A copy of each program on each calculator

## Usage

Run the P1 program on one calculator and P2 on the other. Its a really basic game and not much you can do.
